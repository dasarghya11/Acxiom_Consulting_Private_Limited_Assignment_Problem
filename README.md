Library Management System

Frontend  : JSP + HTML + CSS

Backend   : Java Servlets

Database  : MySQL

Server    : Apache Tomcat(10.1.5)


Library Management System
│
├── Login Module
│
├── Maintenance Module (ADMIN ONLY)
│   ├── Add Book
│   ├── Update Book
│   ├── Add Membership
│   ├── Update Membership
│   └── User Management
│
├── Transaction Module
│   ├── Book Availability
│   ├── Book Issue
│   ├── Return Book
│   └── Fine Payment
│
└── Reports Module
    ├── Issued Books
    ├── Available Books
    └── Members Report
